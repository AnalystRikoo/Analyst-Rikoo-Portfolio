## Projects

### 1. Freelancer Payment Tracker (Excel)

Excel template for freelancers and remote workers to track invoices, payments, and cash flow.

**What it does**:

- Automatically calculates Paid / Overdue / Pending status

- Interactive dashboard with slicers to filter by client or status

- Key metrics: Total Billed, Total Paid, Balance Due

- Charts: Paid vs Outstanding Balance

- Conditional formatting for visual alerts

**How to use**:

1. Download the template below

2. Enter your invoices in the "DATA ENTRY" sheet

3. Go to "DASHBOARD" sheet — slicers and visuals update automatically

4. Refresh PivotTables if needed (right-click → Refresh)
